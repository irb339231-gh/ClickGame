# 10秒間クリックゲーム

10秒間にボタンを何回クリックできるか競うゲームです。
スコアはランキングに保存されます。

# 背景

卒業制作でランキングを導入予定。
ランキングの実装練習。

## デモ

https://irb339231-gh.github.io/ClickGame/

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage（ランキング保存）

## 学んだこと

- HTMLの基本構造
- CSSのflexboxレイアウト
- JavaScriptのDOM操作
- setIntervalを使ったタイマー実装
- localStorageを使ったデータ保存
